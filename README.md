# ChronoChantier - Gestion de Chantier Intelligente

Application de gestion de chantier pour le BTP.

## Installation et Lancement

**Prérequis:** Node.js

1. Installer les dépendances:
   `npm install`
2. Lancer l'application:
   `npm run dev`
